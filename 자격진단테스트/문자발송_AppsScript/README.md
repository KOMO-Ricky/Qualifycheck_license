# 진단표 이미지 문자(MMS) 발송 설치 가이드

신청자가 웹페이지에서 이름·연락처를 입력하면, **입력한 휴대폰번호로 맞춤 진단표 이미지가 문자(MMS)로 발송**되도록 하는 기능입니다.

## 전체 구조

```
[웹페이지(브라우저)]                 [Apps Script 웹앱]              [알리고 / 솔라피]
 진단표 이미지 생성                    doPost()
 → JPG로 압축(≤190KB)      ─────▶   1) 구글 시트 저장     ─────▶   실제 MMS 발송
 → 이름·연락처·이미지 전송            2) 문자 발송 API 호출            → 신청자 휴대폰 도착
```

- **API 키는 절대 웹페이지(HTML)에 넣지 않습니다.** Apps Script "스크립트 속성"에만 저장합니다.
- 웹페이지는 이미지·연락처만 넘기고, 실제 발송은 서버(Apps Script)가 합니다.

## 어떤 파일을 쓰나

| 파일 | 용도 |
|------|------|
| `알리고_Code.gs` | 알리고(Aligo)로 발송 |
| `솔라피_Code.gs` | 솔라피(Solapi)로 발송 |

> **먼저 알리고로 시도**하고, 잘 안 되면 솔라피로 교체하면 됩니다. 둘 다 `doPost` 구조가 같아서 **`.gs` 파일 내용만 통째로 교체하고 다시 배포**하면 전환됩니다.

## 설치 순서

### 1. 웹페이지 교체
`병합결과/개인택시진단_병합.html` 를 실제 서비스 파일로 사용합니다.
(편집 내용 + 문자 발송 이미지 생성·전송 코드가 모두 통합돼 있습니다.
 참고: `_v20260706.html`은 통합 전 SMS 단독 버전으로, 더 이상 사용하지 않습니다.)

### 2. Apps Script 코드 붙여넣기
1. 기존 구글 시트 연동에 쓰던 Apps Script 프로젝트 열기
2. `Code.gs` 내용을 → `알리고_Code.gs`(또는 `솔라피_Code.gs`) 내용으로 교체
   - **이미 시트 저장 코드가 있다면** 전체를 덮어쓰지 말고, `sendAligoMms(data)` / `sendSolapiMms(data)` 관련 함수만 추가하고 기존 `doPost` 안에서 저장 코드 뒤에 호출 한 줄만 넣어도 됩니다.

### 3. 스크립트 속성 등록
편집기 좌측 **⚙ 프로젝트 설정 > 스크립트 속성**에 추가:

**알리고**
| 속성 이름 | 값 |
|-----------|-----|
| `ALIGO_API_KEY` | 알리고 API 키 |
| `ALIGO_USER_ID` | 알리고 아이디 |
| `ALIGO_SENDER` | 발신번호(사전등록된 번호, 예 `15667114`) |
| `SHEET_ID` | 구글 시트 문서 ID |

**솔라피**
| 속성 이름 | 값 |
|-----------|-----|
| `SOLAPI_API_KEY` | 솔라피 API Key |
| `SOLAPI_API_SECRET` | 솔라피 API Secret |
| `SOLAPI_SENDER` | 발신번호(등록된 번호) |
| `SHEET_ID` | 구글 시트 문서 ID |

> `SHEET_ID` = 시트 URL `https://docs.google.com/spreadsheets/d/`**`이부분`**`/edit`

### 4. (알리고만) 발송 IP 제한 해제
알리고 관리자페이지 > **발송 IP 관리 > 허용 IP를 비워둠(전체 허용)**.
→ Apps Script는 구글 서버 IP가 매번 바뀌므로 IP 제한을 켜두면 발송이 실패합니다.

### 5. 웹앱 배포
**배포 > 새 배포 > 유형: 웹 앱**
- 실행 계정: 나
- 액세스 권한: **모든 사용자**
- 배포 후 나오는 **웹앱 URL**을 HTML의 `GAS_WEB_APP_URL` 값과 일치시킵니다.
  (기존에 쓰던 URL을 그대로 재배포하면 HTML 수정 불필요)

### 6. 텍스트 발송 테스트 (권장)
이미지 없이 문자만 먼저 보내 계정·발신번호가 정상인지 확인합니다.
- 알리고: `testAligoText` 함수의 `receiver`를 본인 번호로 바꾸고 편집기에서 실행
- 솔라피: `testSolapiText` 함수의 `to`를 본인 번호로 바꾸고 실행
- 실행 로그(보기 > 로그)에서 성공 응답 확인

### 7. 실제 테스트
웹페이지에서 본인 번호로 신청 → 진단표 이미지 MMS가 도착하는지 확인.
- 브라우저 개발자도구(F12) 콘솔에서 `[DB전송] payload keys ... | image: OOOB` 로그로 이미지 크기 확인 가능

## 참고 / 주의사항

- **MMS는 1통에 이미지 1장**이 규격입니다(알리고·솔라피 공통). 현재는 **진단표 1장**을 발송합니다.
  - 필요 서류 리스트까지 함께 보내려면 (a) 이미지를 세로로 합쳐 1장으로 만들거나 (b) 2통으로 나눠 보내야 합니다. 원하면 추가해 드립니다.
- MMS 이미지 규격: **JPG, 200KB 이하**. 웹페이지가 자동으로 190KB 이하로 압축합니다.
- MMS는 건당 과금됩니다(대략 30~90원). 발송 전 잔액을 확인하세요.
- 발신번호는 **사전등록(통신사 인증)** 된 번호만 사용할 수 있습니다.
- 광고성으로 분류될 수 있는 내용이면 `(광고)` 표기·무료수신거부 안내가 필요할 수 있습니다. 현재 문구는 "요청한 진단표 발송"이라는 정보성입니다.
