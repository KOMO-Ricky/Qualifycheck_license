# 개인택시 자격진단 웹페이지 프로젝트

## 프로젝트 개요
개인택시 자격진단을 위한 단일 HTML 파일 웹페이지. 기능과 디자인을 분리해 개발 후 병합하는 방식으로 작업.

## 파일 구조
```
클로드 코드 작업용/
├── 초안_기능구현/
│   └── taxi_TEST_mobile_v1_69.html      ← 기능 구현 원본
├── 디자인/
│   ├── 개인택시 진단_전체디자인_단독실행.html  ← 디자인 원본
│   ├── 개인택시 진단 디자인소스.html
│   ├── 자격진단 도시에 디자인소스.html
│   └── (이미지 에셋: logo_KOMO.png, 아이콘 등)
└── 병합결과/
    └── 개인택시진단_병합_v20260702e.html   ← 현재 최신 버전
```

## 작업 규칙
- **항상 최신 버전 파일을 기준**으로 작업: `병합결과/개인택시진단_병합_v20260702e.html`
- 수정 후 저장 시 날짜+알파벳 접미사로 버전 관리 (예: _v20260703.html, _v20260703b.html)
- 이미지 에셋은 `디자인/` 폴더에 있으며, HTML 내 Base64 인코딩으로 포함되어 있음
- 모바일 우선 반응형 디자인

## 커뮤니케이션
- 모든 대화는 **한국어**로
- 파일 수정 요청 시 코드 스니펫만 주지 말고 **파일을 직접 수정**할 것
- 변경 사항은 간결하게 요약해서 설명
